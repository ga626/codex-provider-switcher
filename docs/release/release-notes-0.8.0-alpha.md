# Signalman AI 0.8.0-alpha 发布说明

## 本次变化

- 产品显示名称从旧名称更新为 Signalman AI，桌面窗口、安装器、MSIX 显示名、发布资产、README 和 Store 文案同步更新。
- Microsoft Store 的新候选包版本为 `0.8.0.0`，高于此前候选版本；Store ID 和包身份保持不变。
- 保留已有本机资料目录，升级不会丢失 DPAPI 保护的密钥、恢复点或活动记录。
- 修复维护者刷新本机候选安装时，正在运行的桌面进程占用安装文件的问题。

## 发布状态

这是待认证的候选版本，不等于 Microsoft Store 已交付。认证通过后，必须从 Store 安装并完成启动、切换、恢复和更新验收。
