# CodeX Provider Switcher 0.4.0-alpha 发布说明

## 摘要

本版本补齐通用恢复中心和可审计备份，并将下一次正式 Release 作为自动创建、签名更新和跨版本升级的真实验收目标。

## 主要改动

- 安全检查新增最近恢复点展示和恢复前二次确认。
- 切换前备份新增 manifest，只记录创建时间、原因和备份文件名。
- 移除旧工具专用的内部状态、环境变量和自动导入路径；本机替换不再作为产品界面功能。
- 功能分支只依赖本地前置验证与 PR CI，避免与 push CI 重复执行。

## 用户影响

- 用户可以在应用中看到最近恢复点，并在确认后恢复本应用创建的备份。
- 用户无需提供更新签名私钥或口令。
- 恢复后仍需重新检查当前服务商；恢复不代表远端 provider 一定可用。

## 发布后验收

1. 使用新 tag 触发 GitHub Actions，确认 Release 自动创建成功，setup、签名、SHA256 与 latest.json 完整对应。
2. 从 GitHub Release 下载 setup，在现有稳定安装上完成跨版本更新，确认签名校验、重启、版本变化和用户数据保留。
3. 本机实际 provider 替换只在上述验收后，由独立新会话执行，不属于本版本产品界面。
