# 获取帮助

CodeX Provider Switcher 是 Windows 上的本地 provider 管理工具。使用中遇到问题，请优先查看：

- [安装、更新与卸载](docs/user/installation.zh.md)
- [排错指南](docs/user/troubleshooting.zh.md)
- [隐私政策](docs/user/privacy-policy.zh.md)

仍未解决时，请在 [GitHub Issues](https://github.com/ga626/codex-provider-switcher/issues) 新建问题。请提供：

- 应用版本和安装渠道（Microsoft Store 或 GitHub 备用安装包）；
- 可重复的操作步骤；
- 界面中显示的脱敏错误摘要；
- 发生时间和 Windows 版本。

请不要提交 API key、token、`auth.json`、`config.toml`、provider 配置、备份目录、完整日志或包含本机路径的截图。需要私密安全报告时，请按 [SECURITY.md](SECURITY.md) 的方式联系维护者。

项目维护者会先确认问题是否能在不接触用户凭据的条件下复现。GitHub Issues 不是实时客服，也不保证每个第三方服务商问题都能由本项目修复。
